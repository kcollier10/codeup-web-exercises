# HTML Elements Pre Lesson
### Overview
1. In your own words, define an HTML element.
Root element that defines the document type

2. What is a void element?
Never allowed to have contents under any circumstances/ <br>

3. What do all elements begin and end with
Opening and closing tag

4. What is the syntax of an HTML element
<html></html>

5. What do we call an element that contains other elements? What are the nested elements called?
block-level element; inline elements

### Block-Level vs. Inline Elements
1. What is a block level element? What is an example of a block level element?
Occupies the entire space of its parent element(container), creating a 'block' - ex. <div>

2. What is an inline element? What is an example of an inline element?
Occupies only the space bounded by the tags that define the inline element - ex. <em> or <strong>

### Attributes
1. What is an attribute?
additional data inside the element, specified inside the opening tag - ex. <div id="wrapper">
